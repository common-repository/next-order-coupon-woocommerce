Increase your WooCommerce store upsell by emailing customers a coupon once they have completed an order.

== Description ==

This plugin automatically emails customers a single use coupon once they have successfully completed an order in your WooCommerce store.

All you need to do is set the coupon amount & the coupon code, then sit back and enjoy the repeat customers.

See the screenshots page for how the coupon code appears for customers.

Features Coming Soon:

*   Change the style and text of the coupon email
*   Set the coupon's expiring date
*   Set a minimum order amount before an order qualifies for a coupon
*   and more!

== Installation ==

1. Make sure WooCommerce is installed and activated.
1. Upload the plugin files to the `/wp-content/plugins/next-order-coupon` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. (Optional) - Opt in to the Freemius service where plugin updates will be emailed out to customers.
1. Use the WooCommerce Settings->Next Order Coupon screen to enable the plugin and set the coupon code and amount.
1. Enjoy repeat customers to your store!
